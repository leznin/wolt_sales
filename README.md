# Wolt Discounts App

## Описание

Это приложение для отслеживания скидок в сервисе Wolt. Приложение состоит из бэкенда на Flask и фронтенда на React, и использует базу данных SQLite для хранения информации о магазинах и скидках.

## Требования

Для запуска приложения вам потребуется:

- Python 3.8 или выше
- Node.js 14 или выше
- npm (обычно устанавливается вместе с Node.js)
- pip (менеджер пакетов Python)

## Структура проекта

- `app/` - основная директория приложения
  - `server.py` - бэкенд-сервер на Flask
  - `src/` - исходный код React-приложения
  - `public/` - статические файлы
  - `requirements.txt` - зависимости Python
  - `package.json` - зависимости Node.js и скрипты
  - `setup.sh` - скрипт для установки зависимостей
- `database.py` - модуль для работы с базой данных
- `wolt_api.py` - модуль для работы с API Wolt
- `main.py` - основной скрипт для сбора данных
- `wolt_discounts.db` - база данных SQLite с информацией о скидках

## Установка

### Метод 1: Использование скрипта установки

1. Перейдите в директорию приложения:
   ```bash
   cd app
   ```

2. Сделайте скрипт установки исполняемым:
   ```bash
   chmod +x setup.sh
   ```

3. Запустите скрипт установки:
   ```bash
   ./setup.sh
   ```

   Этот скрипт автоматически установит все необходимые зависимости Python и Node.js.

### Метод 2: Ручная установка

1. Перейдите в директорию приложения:
   ```bash
   cd app
   ```

2. Установите зависимости Python:
   ```bash
   pip install -r requirements.txt
   ```

3. Установите зависимости Node.js:
   ```bash
   npm install
   ```

## Запуск приложения

### Запуск бэкенда

1. Перейдите в директорию приложения:
   ```bash
   cd app
   ```

2. Запустите сервер Flask:
   ```bash
   python server.py
   ```

   Сервер запустится на порту 5001 (http://localhost:5001).

### Запуск фронтенда

1. Откройте новый терминал и перейдите в директорию приложения:
   ```bash
   cd app
   ```

2. Запустите сервер разработки React:
   ```bash
   npm start
   ```

   Фронтенд-сервер запустится на порту 3000 (http://localhost:3000).

## Доступ к приложению

После запуска обоих серверов вы можете открыть приложение в браузере по адресу:
- http://localhost:3000

## API Endpoints

Бэкенд предоставляет следующие API endpoints:

- `GET /api/stores` - получить список всех магазинов со скидками
- `GET /api/store/<store_id>/discounts` - получить скидки для конкретного магазина
- `GET /api/top-discounts` - получить список самых больших скидок

## Работа с аккаунтами Telegram

Система использует файлы аккаунтов Telegram, хранящиеся в директории `app/accounts`. Каждый аккаунт представлен двумя файлами:
1. JSON-файл (например, `447386249426.json`) - содержит метаданные аккаунта, включая номер телефона, app_id, app_hash, информацию об устройстве, статус и другие настройки
2. Session-файл (например, `447386249426.session`) - файл сессии Telethon для аутентификации

Путь к директории аккаунтов настроен в конфигурации (`app/backend/utils/config.py`) через параметр `accounts_dir`.

## Устранение неполадок

### Проблемы с базой данных

Если вы столкнулись с ошибками, связанными с базой данных, убедитесь, что файл `wolt_discounts.db` существует в корневой директории проекта. Если файл отсутствует или поврежден, вы можете создать новую базу данных, запустив скрипт `main.py`.

### Проблемы с подключением к API

Если возникают проблемы с подключением к API Wolt, проверьте ваше интернет-соединение и убедитесь, что сервис Wolt доступен.

### Проблемы с фронтендом

Если фронтенд не загружается или отображается некорректно, попробуйте выполнить следующие действия:
1. Очистите кэш браузера
2. Перезапустите сервер разработки React
3. Убедитесь, что бэкенд-сервер запущен и работает корректно# wolt_sales
